Certificate¶

class commandment.models.Certificate(**kwargs)[source]¶

Polymorphic base for certificate types.

These certificate classes are only intended to be used for storing certificates related to running the MDM or certificates issued by the MDM internal CA or SCEP service.

Note that X.509 name attributes have fixed lengths as defined in RFC5280.
